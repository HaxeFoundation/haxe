The alpha of the compiler does only typing, you can only play with it but it will not generate any binary right now

You can also access haXe sources CVS repository using the following : 

    cvs -d:pserver:anonymous@cvs.motion-twin.com:/cvsroot login
    (empty password)
    cvs -s:pserver:anonymous@cvs.motion-twin.com:/cvsroot co haxe

haXe sources are available under the GPL license.
