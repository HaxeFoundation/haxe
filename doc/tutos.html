The haXe Programming Language

Using the compiler

The haXe compiler is a commandline compiler that can take the following parameters :

  • -cp (path) : adds a search path for files. Can be used several times for adding several different paths.
  • -swf (file) : compile code to SWF file (create a new file)
  • -v : verbose mode.
  • ...more later

To compile the class my.Class implemented in the file c:\work\my\Class.hx then simply run : 

    haxe -cp "c:\work" my.Class

Please notice that the haXe compiler only takes class names as input and not file names.

Tutorials

Flash Hello World

Create the following file named Hello.hx in c:\work for example : 

class Hello {
    function new() {
        Boot._global.trace("hello world !");
    }
    static var init : Hello = new Hello();
}

Compile using the haXe compiler : 

    haxe -cp "c:\work" -swf hello.swf Hello

This will produce a hello.swf file. Drag'n drop it to an opened instance of the Flash IDE to see the trace.
